# File Renamer Tool

A simple Python script to rename image and video files in a directory based on their type and sequential numbering.

## Features

* Renames image files with the pattern `IMG-XXX` (e.g., `IMG-001.jpg`).
* Renames video files with the pattern `VID-XXX` (e.g., `VID-001.mp4`).
* Supports common image file extensions: `.jpg`, `.jpeg`, `.png`, `.gif`, `.bmp`, `.tiff`, `.webp`.
* Supports common video file extensions: `.mp4`, `.avi`, `.mov`, `.wmv`, `.mkv`, `.flv`, `.m4v`.

## How to Use

1.  Save the Python script as `file_renamer.py`.
2.  Open your terminal or command prompt.
3.  Navigate to the directory where you saved the script.
4.  Run the script using the command: `python file_renamer.py`
5.  The script will prompt you to enter the path to the directory containing the files you want to rename.
6.  Enter the directory path and press Enter.
